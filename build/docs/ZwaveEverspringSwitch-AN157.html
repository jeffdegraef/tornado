Everspring AN157 Switch

The Everspring AN157 Switch can be added to a ZWave network to wirelessly control devices.

![](images/zwave/ZwaveEverspringSwitch-AN157.png)

Install Z-Wave Stuff

First, you need to buy Z-Wave USB Key that you should plug in any USB available port and some sofware easy to install using the import node menu, as described in the Z-Wave introduction tutorial.

Get the Everspring switch

Get the right one:

![](images/zwave/ZwaveEverspringSwitch-AN157-2.png)

User guide:

evr_an157manual (pdf)

Add the switch to the ZWave Network

  • plug the switch. If it does not blink, you have to exclude it before including it

Exclusion

  • plug the switch, let it in off mode
  • unplug the ZSTick gen 5 USB key
  • push 3 seconds on the ZStick button (it goes bliking orange)
  • toggle the button of the switch 3 times.

Inclusion

  • plug the switch, it blinks red
  • unplug the ZSTick gen 5 USB key
  • push 1s on the button: it blinks blue
  • toggle the swith button 3 times quickly
  • the key button turns solid blue.
