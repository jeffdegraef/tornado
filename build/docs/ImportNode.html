Import Nodes menu

There is a big community that builds software for Node-RED and the Pi. The import Nodes functionality is a dialog that allows to add third party software to the current installation.

Adding new functionalities to TimeSquAir

Use the following menu:

![](images/node-red/ImportNodeMenu.png)

And fill the edit field with the libray name to import

![](images/node-red/import.png)

  • The libraries are published here: http://flows.nodered.org.

  • The libraries dedicated to the Thingbox can be found here: https://www.npmjs.com/search?q=ttb-

Be aware that some nodes need further low level installation (as “apt-get xxx”) that can be only be made by advanced users. These additional steps connaot be guessed by the generic node installer that is described here.

Programmers that publish those packages can learn how to make libraries that work with this installer here: http://thethingbox.io/docs/faq_NodeImport.html.
