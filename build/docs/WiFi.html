Add a WiFi dongle

You can add WiFi funtionalities by adding a dongle in the USB port.

The Thingbox already contains software for the following dongle “Edimax EW-7811UN Nano Adaptator”.

Edit the WiFi Settings

From the Settings option of the right menu, go to the settings page and look at the WifI setting box:

Enter the name of the WiFi network and the password. Hit Ok and then reboot. It should be ok then.
