Backup your flow

Starting with version 1.8.6, there is a button in the settings that allows to download and install new releases in one click.

It does not delete your flow and nodes installed with the import / node menu.

But, it does not update Raspberry Pi system files, Thus the result is not exactly the same as flashing a new SDCard.

If you flash a new SDCard, it is important to save the current work before updating.

Note: You may adapt the first part of the path if you renamed the Thingbox in the settings page.

To save a work from a previous release before thingbox 1.5.0 :

Go on the network explorer (using samba) and search and find the file “flows.json” in the directory 

\\thethingbox\TheThingBox\root\thethingbox\node_modules\node-red\flows\local\flows.json

Replace this file in the new Thingbox.

To save a work from a previous release starting from thingbox 1.5.0 to 1.7.0:

Delete the system tab by double clicking on it (on the 1.8.0, it is no more stored with the user data). Don’t forget to deploy/Activate to send the flow to the Thingbox.

Go on the network explorer (using samba) with a computer and search :

\\\thethingbox\TheThingBox\root\userdir

It will go directly in the directory of the ThingBox where is saved the flow file and user settings.

Copy the userdir folder and paste it somewhere on your pc where it’s easy to recover it (not on the ThingBox).

Follow the different steps to install the new ThingBox.

When it’s done, replace the userdir from the new version by the previously saved version.

Don’t forget to reboot the ThingBox.

To save a work from a previous release starting from thingbox 1.8.0

Go on the network explorer (using samba) with a computer and search :

\\\thethingbox\TheThingBox\root\userdir

It will go directly in the directory of the ThingBox where is saved the flow file and user settings.

Copy the userdir folder and paste it somewhere on your pc where it’s easy to recover it (not on the ThingBox).

Follow the different steps to install the new ThingBox.

When it’s done, replace the userdir from the new version by the previously saved version.

Be aware that local nodes in the userdir may depend of software installed out of the userdir. In this case you may have to re-import them (using the menu import/node).

Don’t forget to reboot the ThingBox.
